# todo-calendar

Application with todo calendar. Day to day planning.

## Prerequisites

Requires:

* [pnpm](https://pnpm.io/)
* [uv](https://docs.astral.sh/uv/)

```shell
cp .env.example .env

make install
make migrate
```

## Run

```shell
make run
```

## Development

### Lint

```shell
make lint
```

### Test

```shell
make test
```

### Frontend

```shell
make dev
```
